## 1. ЗАГАЛЬНА МЕТА

Напівавтоматична система контент-маркетингу для CRMCUSTOMS. Максим пише тільки те, що хоче і коли хоче — кейси, авторські статті, особисті пости. Все інше система робить сама.

**Три головні принципи:**
- Максим витрачає на контент максимум **2-3 години на тиждень**
- Щодня виходить мінімум **1-2 публікації** без його участі
- Система сама себе поповнює: закінчується план → генерує новий → просить затвердити

**Платформи:**
- Telegram канал: [@prodayslonakume](https://t.me/prodayslonakume) («Продай слона | Про бізнес українською»)
- LinkedIn (особиста сторінка або компанія — уточнити)
- Facebook (уточнити)
- Instagram (уточнити)
- Сайт: crmcustoms.com (Netlify + Notion CMS)

**Мова контенту:** виключно українська. Жодних RU-сервісів у контенті.

## 3. МОДУЛЬ 0 — СТРАТЕГІЧНИЙ ФУНДАМЕНТ

Запускається **один раз** на старті (або при зміні стратегії). Результати зберігаються в Notion і служать основою для всіх наступних промптів.

> ⚠️ Без стратегічної бази система генерує «просто контент». З базою — контент, який точно потрапляє в ЦА.

### Ланцюжок промптів

```
Промпт #1 (База/ICP) — запускається ВРУЧНУ через /strategy
  ↓ результат: головна проблема ніші + таблиця ICP

Промпт #2 (7 кутів) — запускається АВТОМАТИЧНО після #1
  ↓ результат: 7 неочевидних кутів з хуками та фактами

Промпт #3 (21 підтема + 3 формати = 63 готових пости)
  ↓ результат: готові пости під Telegram, LinkedIn, Reels

Промпт #4 (Календар 30 днів)
  ↓ результат: JSON → автоматично заповнює Notion контент-план
```

### Де зберігаються результати

| База Notion | Що зберігається |
|-------------|-----------------|
| 🎯 Стратегія / ICP | Головна проблема, таблиця ICP, цілі на 30 днів |
| 📐 Контент-кути | 7 кутів з хуками, фактами, спірними твердженнями |
| 📋 Контент-план | 63 готових пости зі статусами та датами |

## 4. БІБЛІОТЕКА ПРОМПТІВ

Всі промпти зберігаються в Notion (таблиця «Бібліотека промптів»). Щоб покращити якість — редагуєш промпт у Notion, не чіпаючи код.

### 4.1 Стратегічні промпти (ВРУЧНУ)

**[strategy_base] Промпт #1 — Стратегічна база**

```
Ти - контент-стратег. Перед тим, як відповідати, задай мені питання про:
  - ціль, цільову аудиторію, болі, обмеження, тон, платформи

На основі відповідей сформулюй одну головну проблему в моїй ніші,
яку люди реально готові обговорювати і шерити. Дай:
  • чому ця проблема важлива (2–3 чітких речення)
  • цілі контенту на 30 днів (збереження/ліди)
  • таблицю ICP: хто / що хочуть / що їх стопорить /
    як вони говорять / тригери / табу

Без води, конкретно.
```

**[strategy_angles] Промпт #2 — 7 кутів теми**

```
Візьми головну проблему і дай 7 неочевидних кутів:
контрінтуїтивні, "ви робите це неправильно",
міф vs реальність, прихована ціна стандартного рішення.

По кожному: 2 хуки (≤12 слів) + інсайт (≤80 слів) +
факт для довіри + спірне твердження.
Кути не повторюються.
```

**[strategy_posts] Промпт #3 — 21 підтема + формати**

```
По кожному з 7 кутів — 3 підтеми з ефектом
"блін, я про це навіть не думав".

По кожній підтемі — 3 формати:
1. НАВЧАЛЬНИЙ: хук + інсайт + 3 кроки
2. ПРОВОКАЦІЙНИЙ: хук + 3 аргументи + питання
3. КЕЙС: контекст + дії + цифри + висновок
```

**[strategy_calendar] Промпт #4 — Календар 30 днів**

```
По кожному формату адаптуй під Threads / Reels / Telegram.
Постав метрику успіху. Збери контент-календар на 30 днів:
дата → платформа → тема → формат → ціль → дистрибуція.
Без повторів. Чергування тем. Макс 2 продажних/тиждень.
Формат: JSON масив.
```
