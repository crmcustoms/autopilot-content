# Що потрібно для запуску і як працює нейромережа

## Підключені сервіси

| Сервіс | Статус | Де використовується |
|--------|--------|----------------------|
| **Notion** | Підключено | Токен і ID сторінки в .env. Усі бази, контент-план, промпти, куті. |
| **Telegram Bot** | Підключено | @crmcontent_bot, команди /start, /plan, /today, /strategy. |
| **Google AI Studio (Gemini)** | Підключено | 5 ключів у .env (GOOGLE_AI_KEY_1…5). Генерація текстів, strategy_runner, seed_bases_initial. |

## Що поки не підключено (для повного запуску за ТЗ)

| Сервіс | Навіщо |
|--------|--------|
| **Cloudflare R2** | Зберігання зображень/аудіо, CDN. Потрібні ключі в .env. |
| **Replicate** | Генерація зображень (обкладинки, ілюстрації). ID моделі + API key. |
| **Netlify** | Сайт crmcustoms.com, автобілд при оновленні Notion. Build hook. |
| **LinkedIn / Facebook / Instagram API** | Автопублікація постів. Зараз публікація — вручну або через твої n8n-воркфлоу. |
| **Claude (Anthropic)** | Опційно для /strategy замість Gemini. ANTHROPIC_API_KEY у .env. |

**Для поточного запуску (календар, бот, генерація текстів) нічого з цього не обов’язково** — все вже працює на Notion + Telegram + Gemini.

---

## Як зараз працює нейромережа (Gemini)

- **Джерело:** Google AI Studio (Gemini). Ключі з .env (GOOGLE_AI_KEY_1 … GOOGLE_AI_KEY_5).
- **Ротація:** при помилці 429 (ліміт по ключу) скрипт переходить на наступний ключ. Ключі не об’єднуються в один «пул» — кожен має свій ліміт (RPM/TPM) у Google.
- **Токени:** ми їх у коді не рахуємо. Ліміти й витрати видно в [Google AI Studio](https://aistudio.google.com/) (Usage / квоти). Якщо всі 5 ключів впадуть у 429 — генерація не спрацює до скидання ліміту (зазвичай по хвилині/годині залежно від квоти).
- **Де викликається:** strategy_runner (промпт #1 → Notion), seed_bases_initial (стратегія, книги, буферні записи). Бот /strategy запускає strategy_runner.

Токенів зазвичай вистачає для поточного об’єму (один запуск strategy, заповнення баз). Якщо будеш часто генерувати великі об’єми — дивись квоти в AI Studio і при потребі додай ключі або перейди на платний тариф.
